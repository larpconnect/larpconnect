# Security Policy

## Supported Versions

The following versions of LARPConnect are currently being supported with security updates.

| Version             | Supported          |
| ------------------- | ------------------ |
| Current Development | :white_check_mark: |

## Reporting a Vulnerability

We take the security of LARPConnect seriously. If you believe you have found a security
vulnerability, please report it to us by opening a confidential issue on GitHub or by
contacting the maintainers.

Please do not report security vulnerabilities through public GitHub issues.
