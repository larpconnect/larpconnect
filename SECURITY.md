# Security Policy

## Supported Versions

There are no supported versions of this software. This software is **pre-alpha** and thus completely
unsupported. When versions are being supported we will update here.

| Version | Supported          |
| ------- | ------------------ |
| ALL     | :x:                |

Security patches and vulnerability fixes are welcome, however.

## Reporting a Vulnerability

Report security issues through github using the built in security reporting tool, which should be enabled.

You can also email the developers directly.
